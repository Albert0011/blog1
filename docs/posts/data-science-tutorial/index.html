How to start Data Science in 2025: From Zero to Hero

Author

January 18, 2025

In the recent years, data science has always been the top skills that people are getting into. It has been shaping many industries, from finance, healthcare, entertainment, and many more. Maybe you are interested in learning it today? The good news is, in 2025, it gets easier to learn it! In this blog post, I will try to give you a step by step structure on how to start learning data science in 2025. We will cover some key concepts, tools, resources, and learning strategies so that you can be a data scientist yourself!

Why Data Science in 2025?

The world is changing. A lot of advancement in AI has been made this past few years. Data is the key of making AIs. The world is generating more and more data, and as it continues to grow exponentially, more professionals are needed to extact, clean, and make use of these datas. Any kinds of backgrounds can make use data science to make more advancements in their own fields, even if you are from a STEM or non-STEM backgrounds, it doesn’t matter if you have the will!

Where to start?

One of the most frequently asked question is where to start? Here is a good Data Scientist Roadmap that I found, you don’t need to learn everything deeply and everything in one day, but make sure that you know the fundamentals of them. Remember that the most important thing here is your commitment to learning, this journey is not going to be easy, it will take you months to understand everything and just do it at your own pace!

1. Building Your Foundations

a. Maths

Honestly, this is the part where I struggle the most and at first thought that this was not going to be as important as coding. But let me tell you, it is very important in the world of data science. What you need to focus is learning about linear algebra, statistics, and some calculus. What you will learn here are key concepts like hypothesis testing, probability distributions, p-values, regression analysis and many more.

For me personally I like to learn using videos, these videos helped me a lot because it gives some visualizations so that learning math is not too abstract:

  • The Essence of Linear Algebra

  • The Essence of Calculus

  • Mathematics for Machine Learning and Data Science Specialization

Some other materials:

  • A course that I found entertaining

b. Programming

After understanding the math foundations, I will start on learning the essential skill of coding. In 2025, Python is still the most widely used language for data science. This is because it is very simple, easy to pick-up, lots and lots of libraries that are related to data science & AI. A lot of tools for visualization, and many more! Another language is R, but for me personally I don’t really like R because I feel like using python is enough for my day to day basis. R is usually more widely used in research and academia, when you are doing a lot of statistical analysis, then R is definitely prefered.

Here are my suggestions on courses and where to learn for Python:

  • Introduction to Python Programming from Coursera

  • Intro to Programming

  • Kaggle Python Tutorial

  • W3 Schools Python Tutorial

Here are my suggestions on courses and where to learn for R:

  • Data Analysis with R Specialization

  • W3 Schools R Tutorial

c. Database and Data Wrangling

Once you understand the basics of programming, we can go and do a deep dive into the world of databases! I would recommend learning SQL and I think that is going to be enough. What I suggest you to learn is learning PostgreSQL which is open source. Here you will learn a lot about data storage, ability to clean, manipulate data, handling missing values, handling outliers, and many more!

  • SQL Course for beginnners

  • W3 Schools for PostgreSQL

  • Tutorials Point for PostgreSQL

d. Machine Learning

It is not data science if we don’t touch machine learning right? Once you have the basic foundations of programming, data wrangling, statistics, we can now begin the fun part! Machine Learning. By using machine learning, we can learn from data, do predictions, and make decisions. The 3 key concepts that you will need to learn here are Supervised Learning, Unsupervised Learning, and Deep Learning.

Here are some my recommendations of courses:

  • Intro to Machine Learning from Kaggle

  • Intermediate Machine Learning from Kaggle

  • Machine Learning Specialization with Andrew Ng!

  • Machine Learning for Everybody

e. Data Visualization

After you make you gather your data, do some data wrangling, make your ML models, then what’s next? You want to show it to everybody right? Here is where Data Visualization comes in! Data visualization is the art of representing data in a visual format (e.g., charts, graphs) to help uncover patterns, trends, and insights. It’s an essential skill for data scientists, as it allows you to effectively communicate your findings to others. There are a LOT of choices here, for me personally, if we are using python then I would start with Matplotlib + Seaborn if you are from a CS background, and if you are not maybe you can start from Plotly or Altair. For R, ggplot2 is the best one!

Here are some my recommendations but I would suggest to go to their documentations:

  • Fundamentals of Data Visualization by Claus O. Wilke

  • Altair official website

  • Plotly official website

  • Python Seaborn Tutorial

2. Start a project

After finishing all those studies, what I would recommend to do next is just start a project! By doing so you can apply what you have learned so far and get the fruits of it. I usually likes to copy a project that someone has already made and make my own copy of it! Here is some project suggestions to start Data science projects. Another thing that I would do is go through kaggle, click a random project and see how they solve a problem. You can easily copy and see what other people are doing and it will definitely help. Trust me on this, if you don’t understand it, it is okay, just try it, it won’t hurt!

3. Stay Updated!

The world of data science and AI is rapidly evolving, new tools, new libraries, and new researchers are being made everyday, faster than ever. You have to keep stay connected with the updates, read forums online (Reddit is my favourite), join Stack Overflow communities, read news on LinkedIn, Youtube, and many more sources!

Keep learning! Build your portfolio, get that data science job that you have been dreaming about!
